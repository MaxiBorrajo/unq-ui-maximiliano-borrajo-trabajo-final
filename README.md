# TP Final Interfaces

## Requisitos previos

- Node.js
- npm

## Paso a paso

1. **Clonar el repositorio:**

   ```bash
   git clone <URL_DEL_REPOSITORIO>
   cd tp-final-interfaces
   ```

2. **Instalar dependencias:**

   ```bash
   npm install
   ```

3. **Ejecutar en desarrollo:**

```bash
npm run dev
```
